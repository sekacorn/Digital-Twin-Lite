# MVP Scope - Digital Twin Lite

## In Scope

### Inputs (Manual Entry)
- Calories consumed (kcal/day)
- Sleep duration (hours/night)
- Exercise duration (minutes/day)
- Water intake (liters/day)

### Outputs
- Weight trend projection (kg over time)
- Energy score projection (0-100 scale over time)

### Simulation Periods
- 7-day projection
- 30-day projection
- 90-day projection

### Core Features
1. Input form for daily habits
2. Simulation trigger with period selection
3. Line chart visualization of projected trends
4. Summary cards showing predicted outcomes

## Out of Scope (MVP)
- User authentication / accounts
- Historical data tracking
- Machine learning models
- 3D visualization
- Mobile app
- Medical diagnoses or recommendations
- Meal planning or exercise programming
- Integration with wearable devices
