# User Stories - Digital Twin Lite

## Input
- As a user, I want to enter my daily calorie intake so the system can predict my weight trend.
- As a user, I want to enter my sleep hours so the system can predict my energy levels.
- As a user, I want to enter my exercise minutes so the system can factor activity into predictions.
- As a user, I want to enter my water intake so the system can factor hydration into my energy score.

## Simulation
- As a user, I want to choose a simulation period (7, 30, or 90 days) so I can see short and long term projections.
- As a user, I want to run a simulation with my inputs so I can see predicted outcomes.

## Results
- As a user, I want to see a line chart of my projected weight over time so I can visualize the trend.
- As a user, I want to see a line chart of my projected energy score over time.
- As a user, I want to see summary cards with key predicted values (final weight, average energy).

## Disclaimer
- As a user, I want to see a clear disclaimer that this is not medical advice so I understand the tool's limitations.
